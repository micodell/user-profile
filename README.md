## Soal Latihan untuk Final Exam Client-Side Programming using NextJS and Vercel
Buat website profile diri dimana:
- user dapat registrasi yang datanya akan disimpan di Supabase
- user setelah registrasi dan login dapat memasukkan data-data seperti: nama, alamat, No KTP, dan upload foto.
- untuk registrasi dan login gunakan **Server Actions** jangan menggunakan logikan client side
- user dapat update datanya juga
- gunakan typescript
- deploy ke vercel

## Installation
- `pnpm add @supabase/supabase-js @supabase/ssr`
